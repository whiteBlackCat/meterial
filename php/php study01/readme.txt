电影售票项目实践

商品表设计:
  商品名称,图片,价格,原价,库存,描述,颜色
字段:
  int ID,varchar 名称, varchar json 图片,float 价格,float 原价,int 库存,text 描述,varchar json颜色
数据库软件:navicat for MySQL

商品详页情需求:
1.适应手机屏幕
2.显示不同的商品信息
3.访问速度尽可能快

静态数据库类:
单例模式,以减少不必要的数据库连接
 特点:
   私有结构函数防止外部实例化;
   私有克隆函数,防止被克隆.
   静态获取实例函数,自己实例化自己
数据库模型类:
  功能:优化数据库操作,提供数据库操作封装函数
模板类:
  数据与模板结合,获取用户反馈结果
分页类:
  根据输入参数,返回分页的html代码.
